# iOS GaitAuth Sample App

## Getting Started

### Prerequisites

* Xcode 12.0+
* CocoaPods 1.10.0+

### Project Setup

1. Install the locked CocoaPod dependencies.

    ```shell
    pod install --repo-update
    ```

2. Open the workspace file `GaitAuthSample.xcworkspace`.
3. Build and run the `GaitAuthSample` scheme.
